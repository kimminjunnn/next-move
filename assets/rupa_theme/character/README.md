# Rupa Character Assets

The main simulation currently defaults to the SVG stickman character renderer.
The thinner minimal skeleton renderer remains available as a lab/comparison
option. These PNG character directories are still retained for skeleton lab,
legacy views, mascot exploration, and visual rollback.

Do not delete `back/` or the `back-v*` directories just because the product
default moved away from a monkey-forward rig. They are reference and experiment
assets, not debug output.

Only add new generated character assets when a step explicitly asks for them.
Temporary sheets, discarded variants, screenshots, debug outputs, and private
gym photos should stay out of git.
