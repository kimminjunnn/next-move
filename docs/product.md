# Next Move

## Product

Next Move는 실내 볼더링에서 막혔을 때, 벽 사진 위에서 내 몸 기준으로 다음 무브를 직접 시험해보는 볼더링 시뮬레이터 앱이다.

## Core Experience

- 사용자가 벽 사진을 넣는다
- 사진 위에서 손발을 움직여본다
- 다음 동작이 가능한지 직접 가늠해본다

## Not This

- 기록 앱 아님
- 커뮤니티 앱 아님
- AI가 정답만 주는 앱 아님

## V1 Scope

- 홈 화면
- 시뮬레이션 진입 화면
- 프로필 설정
- 벽 사진 입력
- 다음 단계 시뮬레이션 프로토타입

## V1 Exclusions

- 커뮤니티
- 문제 기록/히스토리 중심 기능
- AI 추천 무브
- 복잡한 계정/백엔드
