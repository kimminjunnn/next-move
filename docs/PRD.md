# Rupa PRD

## 제품 정의

Rupa는 실내 볼더링에서 막혔을 때, 사용자가 벽 사진 위에서 자신의 몸 기준으로 다음 무브를 직접 시험해보는 볼더링 시뮬레이터 앱이다.

Rupa는 정답을 대신 내려주는 앱이 아니라, 사용자가 사진 위에서 손발 위치와 자세를 움직이며 “내가 이 무브를 갈 수 있을지”를 가늠하게 돕는 도구다.

## 대상 사용자

- 실내 볼더링 문제를 풀다가 다음 무브에서 막힌 사용자
- 등반 중이거나 직후에 빠르게 동작 가능성을 확인하고 싶은 사용자
- 기록, 커뮤니티, 긴 분석보다 사진 위 직접 시뮬레이션을 원하는 사용자

## 핵심 경험

1. 사용자가 클라이밍 벽 사진을 입력한다.
2. 앱은 사진을 시뮬레이션 화면으로 가져온다.
3. 사용자는 손과 발, 또는 스켈레톤 자세를 사진 위에서 움직인다.
4. 사용자는 다음 동작이 가능한지 직접 판단한다.

## V1 범위

- 홈 화면
- 시뮬레이션 진입 화면
- 프로필 설정
- 벽 사진 입력
- 사진 기반 시뮬레이션 프로토타입
- 스켈레톤 자세 조정
- 벽 사진 분석과 루트 선택을 위한 Expo, Nest API, FastAPI 계약

## V1 제외

- 커뮤니티 기능
- 운동 기록/히스토리 중심 기능
- AI가 정답 무브만 추천하는 경험
- 복잡한 계정 시스템
- 온디바이스 컴퓨터 비전 처리
- 승인되지 않은 사설 암장 사진을 회귀 테스트 fixture로 커밋하는 것

## 성공 기준

- 사용자가 벽 사진을 넣고 시뮬레이션 화면에 진입할 수 있다.
- 사진 위의 스켈레톤 또는 캐릭터 자세가 사용자의 조작에 따라 자연스럽게 바뀐다.
- 좌표계가 명확해서 사진 확대/이동 후에도 오버레이가 어긋나지 않는다.
- route detection 작업에서는 Expo, Nest API, FastAPI 응답 필드가 일관된다.
- 앱 문구는 한국어 중심이며 짧고 직접적이다.

## 비제품 원칙

- Rupa는 사용자의 판단을 대체하지 않는다.
- Rupa는 과한 분석 도구처럼 보이지 않는다.
- Rupa는 SaaS 랜딩 페이지처럼 포장하지 않는다.
- Rupa는 로컬 디버그 산출물과 private 사진을 제품 코드와 섞지 않는다.
