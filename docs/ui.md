# UI Guidelines

## Tone

- 밝고 따뜻한 실내 볼더링장 느낌
- 어두운 분위기 지양
- 과한 SaaS 랜딩 느낌 지양
- 미니멀하지만 차갑지 않게

## Visual Direction

- warm off-white / ivory background
- deep red CTA
- climbing-hold inspired accents
- 필요한 경우 3D hold assets 사용

## Copy Tone

- 한국어 우선
- 짧고 직접적
- 과장된 마케팅 문구 지양
- “직접 시험해본다”는 느낌 유지

## Home Rules

- 홈은 진입용 랜딩
- 핵심 메시지 + 시작하기
- 기능 중복 배치 금지

## Main Rules

- 실제 첫 행동은 사진 입력
- 주기능 / 보조기능 위계 명확하게
