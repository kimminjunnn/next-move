# UI Guidelines

## Tone

- 실제 실내 볼더링장에 들어온 듯한 친근한 느낌
- Rupa는 귀여운 원숭이 클라이밍 파트너처럼 느껴져야 함
- 합판 벽, 피스 구멍, 컬러 홀드, 검은 볼륨의 현장감 유지
- 과한 SaaS 랜딩 느낌 지양
- 기능 앱의 명확함은 유지하되 차갑거나 의료/분석 도구처럼 보이지 않게

## Visual Direction

- 주요 배경은 합판 클라이밍 벽 베이지
- 큰 빈 배경에는 피스 구멍/T-nut 점 질감을 촘촘하고 분명하게 사용
- primary color는 노란 홀드/바나나 계열
- primary 위 텍스트는 흰색보다 짙은 원숭이 브라운 우선
- 파란 홀드/초크백 색은 route/tool 보조 accent로 사용
- 핑크/초록 홀드는 장식 또는 상태 강조에 제한적으로 사용
- 빨강은 primary로 쓰지 않고 삭제/에러 같은 danger에만 사용
- 검은색/짙은 회색은 볼륨, 매트, 사진 캔버스, 오버레이에만 사용
- 필요한 경우 기존 hold asset보다 코드 기반 합판벽/홀드 장식을 우선

## Copy Tone

- 한국어 우선
- 짧고 직접적
- 과장된 마케팅 문구 지양
- “직접 시험해본다”는 느낌 유지
- 브랜드명은 `Rupa`로 통일

## Home Rules

- 홈은 진입용 랜딩
- 핵심 메시지 + 시작하기
- 기능 중복 배치 금지
- 홈 배경은 Rupa 분위기를 가장 강하게 보여주는 화면
- `<HomeHeroHolds />`는 추상 장식보다 실제 클라이밍장 벽처럼 보여야 함
- 구조와 문구를 바꾸기보다 배경, 질감, 색, 장식으로 분위기를 만든다
- 시작 후 내부 화면은 이미지 배경보다 차분한 합판벽 톤을 사용해 안전한 작업 공간처럼 느껴지게 한다

## Main Rules

- 실제 첫 행동은 사진 입력
- 주기능 / 보조기능 위계 명확하게
- 사진/시뮬레이션 캔버스는 가독성이 우선이므로 어두운 작업 영역 유지 가능
- 캔버스 안 action accent는 노란 primary와 파란 route/tool accent를 우선 사용
- 입력/설정 화면은 촘촘한 피스 구멍이 있는 합판벽 배경 + 따뜻한 카드 표면 + 노란 primary/파란 accent 액션으로 통일
